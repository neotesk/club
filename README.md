This is the source-code for the <a href="https://neotesk.club"><kbd>neotesk.club</kbd></a> website.
It uses <a href="https://github.com/preactjs/preact"><kbd>Preact</kbd></a>, powered by
<a href="https://github.com/neotesk/truct"><kbd><img width="9" height="9" alt="logo"
   src="https://github.com/user-attachments/assets/1a8052d1-924c-4799-9037-8a7c6ac5fc68" /> Truct</kbd></a>
and <a href="https://github.com/evanw/esbuild"><kbd><img width="9" height="9" alt="logo"
       src="https://esbuild.github.io/favicon.svg" /> Esbuild</kbd></a>.
This website contains information about who I am, what I do and my works. It's basically a portfolio website, guh.

> [!WARNING]
> This project is highly unstable and contains experimental algorithms. Things might break easily.

### Setup
Set your cwd to your projects directory or whatever you use to store your projects then run this silly command:
```
git clone --recursive https://github.com/neotesk/club
```
After that, make sure <a href="https://github.com/neotesk/truct"><kbd><img width="9" height="9" alt="logo"
   src="https://github.com/user-attachments/assets/1a8052d1-924c-4799-9037-8a7c6ac5fc68" /> Truct</kbd></a>
is installed so you can run build and setup workflows. After you made sure that you have Truct in your system.
install <a href="https://github.com/evanw/esbuild">esbuild</a> and <a href="https://nodejs.org">Node.js</a>
(Skip if you have those installed)

Now that everything is ready, run:
```
truct setup
```

### Build
Run this command to generate static files (build files obviously):
```
truct do
```
